Administrative SMS Notifications

witless.org

witless.org uses SMS only for administrative, operational, and account-related alerts. These messages are not marketing or promotional messages.

What these messages are for

Messages may be used for tool automation alerts, service health warnings, credential or login failure notices, infrastructure notifications, remote access or connectivity issues, and similar system events.

Current use

At present, these SMS alerts are intended for the site owner’s own number only.

How opt-in works

Numbers are manually configured by the administrator inside the relevant tool or workflow. There is no public auto-enrollment form on this site.

Message type

Transactional and administrative alerts only. No advertising, telemarketing, or list sales.

Consent and message disclosures

  • Message frequency varies based on account activity, system conditions, alerts, and automation events.
  • Message and data rates may apply, depending on the recipient’s mobile carrier plan.
  • Reply STOP to opt out.
  • Reply HELP for help.

Important note about enrollment

This website is provided to document the messaging program and related policies. SMS recipients are not added through a public signup form here. Instead, the administrator manually enters an authorized number into a tool configuration after obtaining consent. Numbers are not purchased, scraped, or imported from third-party lists.
